## 🚀 Challenge Overview
This project is part of my **7 Days Data/Programming Challenge**, where I build and share one project every day to improve my skills in Python, data handling, and problem-solving.

## 📌 Project Title
📚 Book Management System (Library System)

## 🎯 Objective
The goal of this project is to build a simple **Book Management System** using Python that can:

- Store book details
- Manage book availability
- Handle issuing and returning of books
- Save and load data using JSON

## 🛠️ Technologies Used
- Python 🐍  
- JSON (for data storage)  

## ⚙️ Features

✔ Add new books  
✔ View all books  
✔ Issue a book  
✔ Return a book  
✔ Save data to file  
✔ Load existing data
